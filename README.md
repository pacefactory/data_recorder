# Node.js TypeScript Project Template

This is a template for creating new Node.js projects using TypeScript. It comes pre-configured with the following features:

- **TypeScript:** For type-safe JavaScript development.
- **Jest:** For unit testing.
- **ESLint:** For code linting.
- **Prettier:** For code formatting.

## Getting Started

To use this template, click the "Use this template" button on the GitHub repository page.

### Installation

```bash
npm install
```

### Available Scripts

- `npm run build`: Compiles the TypeScript code to JavaScript.
- `npm run start`: Starts the application.
- `npm run test`: Runs the unit tests.
- `npm run lint`: Lints the codebase.
- `npm run format`: Formats the codebase with Prettier.
